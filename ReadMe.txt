Tienda de Ropa y Accesorios de The Flash
Descripción
Esta página web es una tienda en línea que ofrece una amplia variedad de ropa y accesorios inspirados 
en el superhéroe The Flash. Desde camisetas con el icónico logotipo hasta accesorios veloces, nuestra
tienda tiene todo lo que necesitas para lucir como tu superhéroe favorito.


Instrucciones de Uso
Visita el sitio web en tu navegador.
Navega por las categorías para encontrar los productos que te interesen.
Esta pagina se encuentra en proceso, asi que la compra aun no esta habilitada.

Requisitos
Navegador web moderno compatible con HTML y CSS.
Tecnologías Utilizadas
HTML
CSS con SASS
Animaciones CSS
Media queries para la adaptabilidad a diferentes dispositivos


Créditos
Inspirado en el personaje de The Flash de DC Comics.
